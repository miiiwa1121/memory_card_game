# Memory Card Game (統合ポータル)

8種類のテーマを持つ神経衰弱（メモリーカードゲーム）を1つのNext.jsプロジェクトに統合したポータルサイトです。
インストール不要・登録不要で、ブラウザから無料で遊べます。

## 収録ゲーム一覧

1. **野菜神経衰弱 (`/vegetable`)**
   野菜の画像ペアを揃えるオーソドックスな神経衰弱です。
2. **色神経衰弱 (`/color`)**
   色の名前とカラーパネルのペアを揃えるゲームです。
3. **言語神経衰弱 (`/language`)**
   日本語と英語のペア（例：「りんご」と「Apple」）を揃えるゲームです。
4. **知識神経衰弱 (`/knowledge`)**
   雑学の問題文と答えのペアを揃えるゲームです。
5. **モールス神経衰弱 (`/morse`)**
   ひらがなとモールス信号のペアを揃えるゲームです。
6. **音階神経衰弱 (`/sound`)**
   カードをめくった時の「音（ドレミ）」を聴き分けてペアを揃えるゲームです。Web Audio APIを使用。
7. **リズム感神経衰弱 (`/rhythm`)**
   音のリズムパターンを聴き分けてペアを揃えるゲームです。

## 技術スタック

- **フレームワーク**: Next.js 16 (App Router)
- **言語**: TypeScript / React 19
- **スタイリング**: Vanilla CSS (CSS Modules)
- **音声制御**: Web Audio API (音階・リズムゲームで使用)

## プロジェクト構成のポイント

- **共通コンポーネントによるUI/ロジックの汎用化**
  画像・色・テキストすべてに対応できる汎用的な `Card.tsx` と `GameBoard.tsx` を実装し、多くのゲームで処理を共通化しています。
- **独立したルーティングとSEO**
  各ゲームは `app/vegetable/` や `app/color/` など独立したディレクトリに配置されており、ゲームごとに個別のメタデータ（タイトル、説明文など）を設定しています。
- **特殊ロジックの安全な分離**
  独自のオーディオ制御を持つゲーム（音階・リズム）は共通コンポーネントを使わず、それぞれのフォルダ内（`app/sound/components/` など）に実装とスタイル（`sound-rhythm.css`）を完全に分離することで、他ゲームへの影響を防いでいます。

## ローカル開発手順

```bash
# パッケージのインストール
npm install

# 開発サーバーの起動
npm run dev
# -> http://localhost:3000 にアクセス
```

## デプロイ (Cloudflare Pages 向け)

このプロジェクトは Cloudflare Pages (Next.js full-stack) へのデプロイを想定しています。
1. GitHub にリポジトリをプッシュ
2. Cloudflare Pages の管理画面で「Gitに接続」を選択
3. **フレームワーク プリセット**: Next.js
4. **ビルドコマンド**: `npm run build`
5. （出力ディレクトリなどの設定はCloudflareのNext.jsプリセットに任せて保存＆デプロイ）
